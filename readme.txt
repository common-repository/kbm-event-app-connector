KBM Event App Connector is a plugin which allows quick access to event's and performances from the Keen Booking Manager System.

== Description ==

The KBM Event App System allows you to display events and booking form's which allows you to accept bookings for events straight from your own wordpress site. 

An account on the KBM is required to be able to use this plugin correctly and obtain the shortcodes required.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `kbm_event_app_connector` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Log into KBM and paste in the generated shortcodes into your content area's.
